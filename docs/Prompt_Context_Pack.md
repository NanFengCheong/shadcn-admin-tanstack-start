## Project

`shadcn-admin-tanstack-start` is a TanStack Start + React + TypeScript admin app using shadcn/ui-style components, Radix UI, TanStack Router, TanStack Query, Tailwind CSS, Vitest, Playwright browser testing, and Cloudflare Wrangler.

## Repo Rules

- Implement one ticket only.
- Do not implement future-ticket features.
- Do not refactor unrelated systems.
- Avoid unnecessary dependencies.
- Keep changes small and verifiable.
- Preserve unrelated dirty work.
- Record out-of-scope issues in `docs/Known_Issues_And_Followups.md`.
